Exercise 0

Load packages

Load the following packages for

  • data wrangling (dplyr), and
  • data visualizaiton (ggplot2)
library(dplyr)
library(ggplot2)

Load data

gap = read.csv("https://bit.ly/gap_data_upd")

Exercise 1

Visualize the relationship between life expectancy and population in 1952 using a scatter plot.

gap52 = filter(gap, year == 1952)

ggplot(gap52, aes(x=lifeExp, y=gdpPercap)) + geom_point()

Exercise 2

Visualize the relationship between life expectancy and population in 1952 using a scatter plot and with size of point reflecting GDP per capita.

ggplot(gap52, aes(x=lifeExp, y=gdpPercap, color=continent, size=gdpPercap)) + geom_point()
